popurlar
========

Link tracker


Setup
=====
Requires OpenResty.

Setup at bottom of `<body>` tag.

    <script src="http://mytrackerpage.com/popurlar.js" async onload="(function() { var track = new Popurlar({ project_id: 1 })();"></script>
